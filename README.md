# Mentorship Backend API

A backend system for a mentorship platform where parents, students, and mentors interact.

## Tech Stack

- Node.js
- Express
- MongoDB
- JWT Authentication

## Features

- User Authentication
- Parent creates student accounts
- Mentor creates lessons
- Students book lessons
- Session tracking
- AI mentor assistant

## Installation

git clone https://github.com/BundeleSourav/Mentorship---Backend.git

cd mentorship-backend

npm install

npm run dev

## API Endpoints

POST /auth/signup

POST /auth/login

GET /students

POST /students

GET /lessons

POST /lessons
